# PasteBin Application

A simple **PasteBin application** built with **React.js**, **Node.js**, and **MongoDB**.

## Features

- Create and store text content (pastes).  
- Set an **expiry time** for each paste.  
- After expiry, the paste content will **no longer be displayed**.  
- Simple and user-friendly interface.

## Tech Stack

- **Frontend:** React.js  
- **Backend:** Node.js (Express)  
- **Database:** MongoDB

## How to Run

### Backend
1. Go to the backend folder:
   cd backend
### frontend
1. Go to the backend folder:
   cd frontend
### Install dependencies
1. npm install
2. npm start
3. npm run dev - frontend
